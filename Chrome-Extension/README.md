# AutoForm Fill (Chrome)

## Install

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. **Load unpacked** → select this folder

## Use

- Popup → **Fill form**
- Shortcut → `Ctrl+Shift+V` twice on the page
- **Settings** → Options from popup or extension menu

## Settings

- **General** — shortcut, phone format, custom upload files
- **Fields** — turn field types on/off
- **Rules** — pattern → fill type (or skip)

For testing only. Do not submit fake data on real sites.
